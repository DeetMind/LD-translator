Method Notes

  • This is a translation prototype, not a full catastrophe model. It does not generate new event scenarios.
  • It applies a physical mitigation logic to an existing event set (your uploaded CSV) to produce adjusted losses.
  • The purpose is to show how a flood-control intervention changes insurance-relevant metrics: EAL, EP curve, PML, layer pricing, and capital exposure.
  • Results are indicative only. Consult a licensed actuary or cat model vendor for binding decisions.
